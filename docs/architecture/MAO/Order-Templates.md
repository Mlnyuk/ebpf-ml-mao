# 에이전트별 오더 템플릿

이 문서는 각 에이전트에 작업을 보낼 때 바로 사용할 수 있는 템플릿 모음이다.

## 1. 공통 오더 템플릿

```text
목표:

범위:

제외:

산출물:

검증:

제약:
```

## 2. Claude Code용 템플릿

```text
다음 작업에 대한 설계 초안을 작성해라.

목표:
{goal}

범위:
{scope}

제외:
{non_goals}

원하는 출력:
- 핵심 구조
- 인터페이스 초안
- 리스크 3개
- 구현 전에 고정해야 할 결정사항
```

## 3. Gemini CLI용 템플릿

```text
다음 구현 작업에 대해 탐색/검토해라.

목표:
{goal}

현재 구현 가정:
{current_state}

찾을 것:
- 유사 구현 패턴
- 파싱/운영 리스크
- 테스트 우선순위
```

## 4. Codex용 템플릿

```text
이 저장소에서 바로 구현해라.

목표:
{goal}

범위:
{scope}

제외:
{non_goals}

산출물:
{deliverables}

검증:
{verification}

추가 조건:
- 필요한 경우 디렉터리 구조도 같이 정리
- 테스트와 문서까지 같이 반영
- 완료 후 변경 결과를 요약
```

## 5. Judge 체크 템플릿

```text
다음 항목을 기준으로 완료 여부를 판단해라.

- 목표와 산출물이 일치하는가
- 코드와 문서 경로가 일치하는가
- 테스트가 통과하는가
- README 또는 단계 문서가 갱신되었는가
- 다음 단계로 넘어가도 되는가
```
